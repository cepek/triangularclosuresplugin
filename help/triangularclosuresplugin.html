Triangular Closures Plugin

Extension of Gama-q2 for excluding outlying angular measurements in triangles based on Normal distribution test.

Quick Start

Select or deselect some measurements, which should not be used in adjustement. Outlaying coupples are marked by red text-color.

File menu

This menu contains some basic functions.

Close

Close this plugin.

Settings menu

It allows to you customize settings here.

Reset

Data are cleared from the table and reloaded from Gama-q2. Precission were reseted, too.

Set All Active

It sets all measurements (angles and directions) in local network active.

Set All Passive

It sets all measurements (angles and directions) in local network passive.

Sort By Quality Asc

Sort by ratio of normal distribution test and absolute value of closure in ascending order.

Sort By Quality Desc

Sort by ratio of normal distribution test and absolute value of closure in descending order.

Directions

Row filter for excluding or including directions.

Angles

Row filter for excluding or including angles.

Unused

Row filter for excluding or including used or unused measurements.

Standard Deviation

Column filter for showing or hidding standard deviations.

Type of Measurements

Column filter for showing or hidding type of measurements.

Normal Distribution Test

Column filter for showing or hidding Normal Distribution Test.
